
# Awaken
Awaken is a Hack 'n' Slash game made with Phaser


## How to play
### Online
- Open [Demo](http://clementbaconnier.com/awaken)

### Offline
- Run `php -S localhost:8080` in `/`
- Browse `localhost:8080`
